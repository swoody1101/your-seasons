# Front-end

### 기술스택
  - FE : React 18, JavaScript, HTML5, Material UI
  - 라이브러리
    - Redux toolkit
    ```
      npm install @reduxjs/toolkit
    ```
    - Redux Persist
    ```
      npm install redux-persist
    ``` 
    - React Router
    ```
      npm install react-router-dom@6
    ```
    - Material UI
    ```
      npm install @mui/material @emotion/react @emotion/styled
    ```
